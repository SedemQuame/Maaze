# Maaze
Maaze is a combination of action shoot games and puzzle games. Where players have to complete a set of objectives, to advance in level. Task difficulty becomes increases as the maze expands and smarter, faster and stronger enemies and the maze begins to develop a mind of it's own.

Instructions
One pc or web use [WASD] to move, [IJKL] to rotate and left click to shoot.
All other instructions will be given to you in-game by the objective manager.
For a great experience, please play on your computer, or download the Android APK, if you use an android phone.

Deployed Versions
Android APK can be downloaded from the PlayStore https://play.google.com/store/apps/details?id=com.seddieGames.Maaze
